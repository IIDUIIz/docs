## O que é a API do Hadem

A API do Hadem permite enviar mensagens para seus contatos diretamente dos seus sistemas — CRM, automações, plataformas internas — sem precisar acessar o painel.

## 1. Crie um app e copie o token

No painel, acesse **Settings → Developers** e crie um novo app. O token é exibido uma única vez na criação — guarde-o com segurança.

<Warning>
  Não é possível recuperar o token depois. Se perder, gere um novo em **Settings → Developers → Rotate token**.
</Warning>

## 2. Autentique suas requisições

Inclua o token em todas as chamadas usando o header `Authorization`:

```bash
Authorization: Bearer hm_live_xxxxxxxxxxxx
```

## 3. Faça sua primeira chamada

```bash
curl -X POST "https://api.hadem.com.br/v1/channels/SEU_CHANNEL_ID/messages" \
  -H "Authorization: Bearer hm_live_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "contact": { "value": "5511999999999", "type": "phone" },
    "message": {
      "type": "text",
      "text": { "value": "Olá! Mensagem enviada via API." }
    }
  }'
```

Uma resposta `202` confirma que a mensagem foi aceita e será entregue.
