---

<Note>
  **GET** `https://api.hadem.chat/v1/channels`
</Note>

## Autenticação

<Snippet file="snippets/auth.mdx" />

## Exemplos

<CodeGroup>

```bash cURL
curl "https://api.hadem.chat/v1/channels" \
  -H "Authorization: Bearer hm_live_xxxxxxxxxxxx"
```

```js JavaScript
const res = await fetch("https://api.hadem.chat/v1/channels", {
  headers: {
    Authorization: `Bearer ${process.env.HADEM_API_TOKEN}`,
  },
});

const channels = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://api.hadem.chat/v1/channels",
    headers={"Authorization": f"Bearer {HADEM_API_TOKEN}"},
)

channels = res.json()
```

</CodeGroup>

## Erros

| Status | Descrição |
|--------|-----------|
